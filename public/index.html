API Endpoints at a Glance

Hashing & Authentication

Endpoint Category Description
/hash Hashing Hash a message or file content (e.g., for password storage).
/compare-hash Hashing Verify message integrity by comparing a raw message against a stored hash.
/hmac HMAC Generate a cryptographic hash function with a secret key to authenticate messages.

Encryption & Decryption

Endpoint Category Description
/encrypt Symmetric Encrypt plain text data using the high-performance AES algorithm.
/decrypt Symmetric Decrypt AES-encrypted ciphertext back to readable text.

Digital Signatures & Keys

Endpoint Category Description
/generate-keypair Asymmetric Securely create a new public/private RSA key pair.
/sign Asymmetric Generate a digital signature for a message using a private key.
/verify-signature Asymmetric Authenticate a message using the sender's public key and the digital signature.

Utilities & Encoding

Endpoint Category Description
/encode Utilities Encode raw data into standard formats like Base64 or Hex.
/decode Utilities Decode data from Base64 or Hex back to its original format.
